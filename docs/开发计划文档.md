# 智慧中医医院试点项目管理平台 - 开发计划文档

## 一、项目概述

### 1.1 项目背景

智慧中医医院试点项目管理平台是一个面向 **医院、省局、国家局、专家、系统管理员** 5 类核心角色的全生命周期管理系统，旨在实现智慧中医医院试点项目的数字化、标准化、智能化管理。

### 1.2 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 2.7.x |
| ORM | MyBatis-Plus 3.5.x |
| 工作流 | Flowable 6.x |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis |
| 前端框架 | Vue 3 + TypeScript |
| UI组件 | Ant Design Vue 4.x |
| 表单/表格 | Vben Form/Vxe Table |
| 构建工具 | Vite |

### 4.1 指标管理模块 (Indicator) ✅ 已完成

#### 4.1.1 功能说明

指标是平台的核心数据模型，用于定义各类业务填报字段。主要功能包括：

- 指标的 CRUD 操作
- 按项目类型、业务类型筛选指标
- 指标值类型支持（数字、字符串、布尔、日期、文件上传等）
- 指标逻辑校验关系配置
- 指标列表展示控制

```typescript
function renderIndicator(indicator: DeclareIndicatorApi.Indicator) {

    case 5: // 长文本

    case 6: // 单选

    case 7: // 多选
```

## 七、常见问题处理

### 7.1 指标值存储

不同值类型的指标，存储方式不同：

```typescript
// 根据 valueType 存储到不同字段
switch (indicator.valueType) {
  case 1: // 数字 -> valueNum
    item.valueNum = String(value);
    break;
  case 2: // 字符串 -> valueStr
  case 6: // 单选 -> valueStr
  case 7: // 多选 -> valueStr (JSON)
  case 9: // 文件上传 -> valueStr (JSON)
    item.valueStr = String(value);
    break;
  case 3: // 布尔 -> valueBool
    item.valueBool = value;
    break;
  case 4: // 日期 -> valueDate
    item.valueDate = value.format('YYYY-MM-DD HH:mm:ss');
    break;
  case 5: // 长文本 -> valueText
    item.valueText = String(value);
    break;
  case 8: // 日期区间 -> valueDateStart/valueDateEnd
    item.valueDateStart = value[0].format('YYYY-MM-DD HH:mm:ss');
    item.valueDateEnd = value[1].format('YYYY-MM-DD HH:mm:ss');
    break;
}
```

使用 `v-if` 或计算属性动态渲染不同类型的输入组件：

```vue
<a-switch v-else-if="indicator.valueType === 3" ... />

```
